What is Photon™

Photon™ is a technology preview of a minimal Linux container host. It is designed to have a small footprint and boot extremely quickly on VMware platforms. Photon™ is intended to invite collaboration around running containerized applications in a virtualized environment.

  • Optimized for vSphere - Validated on VMware product and provider platforms.
  • Container support - Supports Docker, rkt, and the Pivotal Garden container specifications.
  • Efficient lifecycle management - contains a new, open-source, yum-compatible package manager that will help make the system as small as possible, but preserve the robust yum package management capabilities.

See the FAQ for more information.

Getting Photon™

If you want a quick and easy way to run Photon™, you can grab a convenient, pre-built ISO image here. The ISO images are distributed under the VMware Technology Preview License Agreement. Open source license information may be found in the Photon™ Open Source License file.

Photon™ source code is available on the VMware Photon™ GitHub source repository. You can build your own Photon™ ISO image by cloning the repo and following the instructions in README.md.

Support

Photon™ is released as open source software and comes with no commercial support.

But since we want to ensure success and recognize that Photon™ consumers might fall into a range of roles - from developers that are steeped in the conventions of open-source to customers that are more accustomed to VMware commercial offerings, we offer several methods of engaging with the Photon™ team and community.

For our developer community, feel free to join our Google groups at:

  • vmware-photon-dev
  • vmware-photon-announce

For more general user questions, visit the Photon™ user forum in our Photon™ VMware Community.
